# project-tune-traveller

A SvelteKit application for planning music-inspired travel experiences.

## Developing

Install dependencies and start the local development server:

```sh
npm install
npm run dev
```

To open the app in a browser automatically:

```sh
npm run dev -- --open
```

## Quality checks

```sh
npm run lint
npm run check
```

## Building

Create a production build with:

```sh
npm run build
```

Preview the production build locally with `npm run preview`.
